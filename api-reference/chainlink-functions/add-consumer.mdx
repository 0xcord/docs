---
title: Functions Add Consumer
description: API endpoint for adding a consumer to the Chainlink Functions subscription
api: "POST https://0xcord.com/api/chainlinkFunctions/addConsumer"
---

## Params

<ParamField query="network" type="string" required>
  The blockchain network where the consumer will be linked (e.g.,
  polygon_mumbai, fuji, sepolia)
</ParamField>
<ParamField query="consumer" type="string" required>
  The address of the consumer contract to be linked
</ParamField>

## Response

<ResponseField name="data" type="object">
  <ResponseField name="success" type="boolean">
    Indicates whether the consumer was successfully linked
  </ResponseField>
  <ResponseField name="transactionHash" type="string">
    The transaction hash of the linking transaction
  </ResponseField>
  <ResponseField name="url" type="string">
    The URL to the transaction details on the blockchain explorer
  </ResponseField>
</ResponseField>
